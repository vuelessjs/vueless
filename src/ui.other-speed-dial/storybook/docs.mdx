## Usage

The `USpeedDial` component is a floating action button (FAB) that expands to reveal multiple quick actions. It supports flexible positioning, smooth animations, and full accessibility.

<Canvas of={stories.Default} />

## Direction

Control the direction in which action items expand using the `direction` prop. Available options: `up`, `down`, `left`, `right`, and `circle`.

<Canvas of={stories.Direction} />

## Size

Control the size of the speed dial using the `size` prop.
