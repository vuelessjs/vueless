## Control loader
You can control loader state using *useLoaderRendering* composable.

<Source code={`
import { useLoaderTop } from "vueless";

const { 
    setLoaderOn,
    setLoaderOff,
    addRequestUrl,
    removeRequestUrl
} = useLoaderTop();
`} language="jsx" dark />

## Control outside of component
You can control loader state outside of vue components using following methods.

<Source code={`
import {
  loaderTopOn,
  loaderTopOff,
} from "vueless";
`} language="jsx" dark />
